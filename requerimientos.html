¿Que necesitas para usar nuestra app?

.


Requerimientos del Sistema

  • 📷 Tener una cámara web o usar la cámara del celular.
  • 🌐 Contar con una conexión a internet estable (requerida para futuras versiones en línea del traductor).
  • 🧭 Usar un navegador actualizado (Chrome, Edge o Brave).
  • 🎥 Permitir el acceso a la cámara cuando el sitio lo solicite.

Requerimientos del Usuario

  • ✋ Realizar las señas de forma clara y frente a la cámara.
  • 💡 Estar en un lugar bien iluminado para una mejor detección.
  • 📖 Tener conocimientos básicos de lectura en español para comprender las traducciones.
  • 📖 Tener conocimientos básicos sobre Lengua de Señas Colombiana (LSC) para usar el traductor.
